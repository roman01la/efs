Rectangular Waveguide (WR42)

WR42 rectangular waveguide with TE10 mode, 20-26 GHz. Port of the openEMS Python tutorial.

About this example

A WR42 rectangular waveguide (10.668 × 4.318 mm cross-section) operating in TE10 mode from 20 to 26 GHz. Two rectangular waveguide ports are used for S-parameter analysis. The simulation verifies the waveguide impedance against the analytic TE10 formula: ZTE = Z0 / √(1 - (fc/f)²). This is a port of the openEMS Python tutorial Rect_Waveguide.
