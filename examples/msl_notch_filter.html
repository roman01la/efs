MSL Notch Filter

Microstrip line notch filter on RO4350B substrate with an open stub. Port of the openEMS Python tutorial.

About this example

A 600 μm microstrip transmission line on RO4350B substrate (εr=3.66) with a quarter-wave open stub that creates a notch (band-reject) filter. Two MSL ports are used to compute S11 (return loss) and S21 (insertion loss). The notch frequency depends on stub length and substrate properties. This is a port of the openEMS Python tutorial MSL_NotchFilter.
